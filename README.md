## Adaptable tutor with Agentic AI

### Instructions

#### 1. git clone https://github.com/akshanshs/AgenticTutor.git

#### 2. create .env and put 
OPENAI_API_KEY=, 
export LANGSMITH_TRACING=true
export LANGSMITH_ENDPOINT=
export LANGSMITH_API_KEY=
export LANGSMITH_PROJECT=

#### 3. cd AgenticTutor --> uv sync

#### 4. Download the vector data (vector_physics) from: https://drive.google.com/drive/folders/1pSvw7dQ4OTxMiwWvD2JJOlXq6rObCCwu?usp=sharing

#### 5. Put vector_physics in vector_db/

#### 6. uv run streamlit run app.py

#### 7. Learn
